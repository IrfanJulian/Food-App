## Installation

Install my-project with npm

```bash
First you can clone my project with git: git clone https://github.com/IrfanJulian/Food-App.git
```
    
Go to the project directory

```bash
  cd my-project
```

```bash
  npm install
```

Start the server

```bash
  npm run start
```
